• Originally built a couple years back for Comcast, Xfinity.com marketing pages.
  • UX request was custom design of controls area.
  • My POC was used for v1 of the web accessible video player. My idea was to bind fake controls to hidden input elements for full keyboard functionality support.
  • Toggle controls and advance vid and volume using keyboard spacebar / arrow keys.
  • Can also plug in a different vid timestamp in the circular time counter above the time slider handle. Example... type in ":02" into the time input.
